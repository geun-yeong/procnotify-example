# procnotify-example

We can monitor for creating or terminating processes and threads and loading images. This repository has basic usages of CreateProcessNotify, CreateThreadNotify and LoadImageNotify.

- procnotify: This project has basic usages of CreateProcessNotify, CreateThreadNotify and LoadImageNotify.
  - This project shows what processes were created or terminated, what threads were created or terminated and what images were loaded.
